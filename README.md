# MPic
在Windows系统下，利用七牛云存储作为图床，拖曳上传图片生成MarkDown格式图片链接的图床神器
>1、选择上传
>2、拖曳上传
>3、截图上传
>4、复制上传

![MPic](http://image.lzhaofu.cn/blog/2016-09-21/220347339)

![设置七牛云存储账号](http://image.lzhaofu.cn/blog/2016-09-19/092425728)

注意：
>1、使用前请先设置七牛云存储账号

>2、同样存储空间也改为自己七牛存储空间的名字
